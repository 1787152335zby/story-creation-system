# 首页重设计 — 设计文档

## 概述

将首页从功能型布局重新设计为高级感叙事型布局，保留所有现有功能不变，仅重构 UI 结构和样式。

## 布局结构

```
┌──────────────────────────────────────────────┐
│  Hero 区（大卡片）                            │
│  - 「✨ AI 驱动全流程」标签                    │
│  - 「从灵感到银幕」标题                        │
│  - 描述文案                                   │
│  - 「🎬 开始创作」按钮                        │
│  - 统计：已完成项目 / 创作环节数 / AI 模型数   │
├──────────────────────────────────────────────┤
│  创作工具（4 张卡片网格）                     │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐       │
│  │新建  │ │生图  │ │视频  │ │剧本  │       │
│  │项目  │ │      │ │生成  │ │目录  │       │
│  └──────┘ └──────┘ └──────┘ └──────┘       │
├──────────────────────────────────────────────┤
│  最近项目（项目列表）                         │
│  搜索栏 + 状态筛选                           │
│  项目卡片网格（含进度条）                     │
│  「查看全部」按钮                             │
├──────────────────────────────────────────────┤
│  创作模板（如有）                             │
└──────────────────────────────────────────────┘
```

## Hero 区

- 渐变背景（primary 色 + accent 色，低透明度）
- 右上角装饰性 radial gradient 光晕
- 标签徽章（`✨ AI 驱动全流程`）
- 主标题 `从灵感到银幕`（28px font-bold）
- 描述文本（14px, muted-foreground, max-w 500px）
- 渐变按钮 `🎬 开始创作`
- 数据统计条：已完成项目数 / 创作环节数 / AI 模型数（数据从现有 state 获取）

## 创作工具卡片

保留现有的 4 个 entry 卡片（新建项目、智能生图、视频生成、剧本目录），视觉效果优化：
- 卡片背景 `#1a1525` → 改用 `glass-card` 类（保持现有主题变量一致）
- hover 提升效果：`translateY(-4px)` 微上浮 + 边框发光
- 图标区稍微加大（44px → 从目前 40px 微调）
- 保持现有 `card-hover` 交互逻辑

## 最近项目区

保留现有项目列表的全部交互功能：
- 搜索过滤
- 状态筛选（全部/进行中/已完成）
- 展开收起
- 删除确认弹窗

布局从 3 列网格改为和卡片网格一致风格，每张项目卡片显示：
- 项目名称
- 类型 + 日期
- 阶段标签（带颜色区分已完成/未完成）
- 进度条（保持现有 progress-glow 样式）
- hover 时显示操作按钮（打开文件夹/删除）

## 技术变更

- 仅改 `src/pages/HomePage.tsx` 一个文件
- 保留全部 state / effect / handler 逻辑，只重构 JSX 渲染部分
- 不新增依赖
- 保留所有交互功能（搜索、筛选、删除、模板列表等）

## 实施范围

只改 HomePage.tsx，不改其他文件。
